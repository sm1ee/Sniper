What is Sniper?

Sniper is an open-source MITM proxy that intercepts and modifies HTTP/HTTPS traffic between your browser and the internet. Built in Rust as a native macOS app, it provides the core workflow for web application security testing, penetration testing, and bug bounty hunting — without the overhead of Java-based tools.

Features

HTTP/HTTPS Capture

Full MITM proxy with persistent root CA. Inspect every request and response in real time. WebSocket support included.

Intercept & Modify

Hold requests before they reach the server. Edit headers, body, and parameters on the fly. Auto match & replace rules.

Passive Scanner

Automatically detect sensitive data leaks, CORS misconfigurations, missing security headers, JWT weaknesses, and more.

Replay

Resend any captured request with modifications. Override target host and port. Follow redirects.

Fuzzer

Mark insertion points and run payload lists. Test for injection vulnerabilities with a simple marker-based interface.

CLI & AI Automation

JSON-first CLI for scripting. Built-in Claude and Codex skill templates let AI agents drive the full workflow.

Quick Start

  1. Download and open Sniper.dmg
  2. Set your browser proxy to 127.0.0.1:8080
  3. Visit https://sniper to trust the root CA
  4. Start capturing traffic

Or build from source:

git clone https://github.com/sm1ee/Sniper.git
cd Sniper
cargo run --bin sniper-desktop
